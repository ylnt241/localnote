# ✅Сайт для заметок и задач✅

Данный сайт обязательно понравится вам за счёт:

разнообразного функционала, милого внешнего вида и, конечно, удобства✨

Разберём каждый пункт.

1. 💟Дизайн сайта для настоящих мужчин, ведь в нём есть всё, что они любят: 
розовый цвет и котики. 

1. Чтение списка дел или бытовых заметок теперь будет расслаблять и радовать вас!

2. 🕶С готовыми заметками можно производить самые необходимые операции:

выбор нескольких, удаление, закрепление и экспорт в Anki.

3. 📔Также вы можете сортировать заметки разными способами:

по алфавиту, по дате обновления, по приоритету!

4. 🎨Конечно же сами заметки вы также можете настроить по своему желанию!

5.   Присвоить тег, цвет и даже тематическую иконку!

6. 👩‍🏫Ещё вы можете активировать особые функции:

    - перевод слов в заметке на английский, чтобы изучать язык на бытовых примерах и научиться рассказывать о том, что вы реально делаете и чем живёте.


## Стек проекта:

* Язык: TypeScript.

* Библиотека интерфейса: React.

* Стилизация: Tailwind CSS.

* Сборка: Vite.

* Хранение данных: LocalStorage.

## Запуск:

```bash
git clone https://github.com/ylnt241/localnote.git
cd localnote/note-site
npm install
npm dev run
```
